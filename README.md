# Conway-s-game-of-life-clone
COMPILE:

	Linux: gcc main.c -o game -lGL -lglut -lGLU
CONTROLS:

	Use the mouse to change the state of a cell.
	Use the 'N' key to compute and next generation.
	Use the 'C' key to reset the state of all cells.
